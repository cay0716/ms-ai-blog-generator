# MS AI Blog Generator
| openAI 기반 개발 블로그 자동 생성 및 커스터마이징 에디터 웹 애플리케이션 |
**[배포 주소](https://ms-ai-blog-generator.vercel.app/)**

## 🚀 프로젝트 소개
MS AI Blog Generator는 AI 기반 콘텐츠 생성과 사용자 편집 경험을 결합한 웹 서비스입니다.

생성된 Markdown 콘텐츠를 즉시 수정할 수 있는 에디터 환경을 제공하며,
코드 블록 하이라이팅 및 다크모드 대응을 통해 실제 기술 블로그 작성 환경과 유사한 경험을 구현했습니다.

**제공되는 글 템플릿**
- 튜토리얼
- TIL
- 트러블 슈팅

**제공되는 작성 스타일**
- 친절한 설명형(~해요, ~입니다)
- 간결한 개발자 스타일(~함, ~임)
- 개발 일지 스타일(~했다, ~이다)
